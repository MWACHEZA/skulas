# Skulas Test Credentials

This document contains all the automatically seeded schools, plans, and user accounts available for testing in the database.
Each school has been seeded with rich, intertwined data including subjects, teachers, students, clinic staff & medical records, ancillary staff, parents, suppliers, alumni, assignments, fees, grades, and attendance tailored to its specific type.

All passwords follow the role name capitalized followed by `@1234` (e.g. `Admin@1234`, `Clinic@1234`, `Student@1234`, `Teacher@1234`, `Ancillary@1234`, `Parent@1234`, `Supplier@1234`, `Alumni@1234`).
