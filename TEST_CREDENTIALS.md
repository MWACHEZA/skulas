# Skulas Test Credentials

This document contains all the automatically seeded schools, plans, and user accounts available for testing in the database.
Each school has been seeded with rich data including subjects, teachers, students, assignments, fees, grades, and attendance tailored to its specific type.

All passwords are the role name capitalized followed by `@1234` (e.g. `Admin@1234`, `Student@1234`, `Teacher@1234`, `Ancillary@1234`). 

> **Note:** Each school now seeds 12 students (`student1` to `student12`) and various staff/teachers utilizing secondary roles (e.g., `sportsmaster@...`, `agriteach@...`, `receptionist@...`, `matron@...`, `tuckshop@...`, `driver@...`, `alumni@...`). You can use these to test the different portals and secondary role permissions.
